# Commands

**\_bwaa** [count]\
Makes the bot type :neuroBwaa: or :evilBwaa:, inputting a value for `count`
will determine the number of bwaa's in the message.

**\_help**\
Displays a list of commands.\
(Command not yet implemented)

**\_restart**\
Restarts the bot.
(Only the owner of the bot can use this command)\
(Only works if running the bot with `firecrafterbot.service` otherwise
it will just stop the process)

**\_stop**\
Stops the bot.\
(Only the owner of the bot can use this command)

**\_save**\
Saves the bot data.\
(Only the owner of the bot can use this command)

**\_reload**\
Reloads the bot data.\
(Only the owner of the bot can use this command)

**\_toggleAutosave**\
Disables/enables autosaving of the bot data.\
(Only the owner of the bot can use this command)\
(Does not persist through restarts)

**\_override**\
Allows for setting a variety of overrides for values!\
for a guide just run the command!

**\_fish**\
Makes the bot fish.

**\_points**\
Displays the user's total points.

**\_leaderboard**\
Displays the points leaderboard.
